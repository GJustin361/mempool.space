  • FAQ What is Mempool?

    Mempool is a block explorer for Bitcoin and some of its Layer 2 networks like Liquid and Bisq.

    Mempool is not a Bitcoin mining pool. Mempool is also not a wallet.

    Like many other block explorers, Mempool merely provides data about the Bitcoin network. It does not "control" the Bitcoin network in any way, and it cannot help you make or change Bitcoin transactions.

         What can I do on the Mempool website?

    Mempool's real-time metrics and visualizations can help you understand the Bitcoin network in a variety of ways.

    Some examples:

    • see status of your Bitcoin transactions
    • see how long a transaction may take to confirm
    • get suggestions for fee-rate (s/vb) based on how quickly you need a transaction to confirm
    • see Layer 2 Bitcoin network metrics (e.g. Liquid, Bisq)

    Much of Mempool's data is available freely for casual use through an API.

         I'm having issues with a transaction. It's taking a very long time to confirm, it got sent to the wrong address, or Mempool can't find it at all. What can I do?

    If you need support for a specific transaction, you must work with the entity who helped you make that transaction (wallet software, exchange company, etc). Mempool can only help you see data about these transactions.

         Can I self-host Mempool?

    Yes! Mempool is free/libre open-source software, and it's built to make users sovereign so they don't have to trust anyone.

    You can host your own Mempool instance in a variety of ways for a variety of use cases. Many people find it easiest to run Mempool on a full node solution like Umbrel, RaspiBlitz, and myNode. Depending on your needs, you can also run Mempool on a custom Linux or FreeBSD setup.

    See more details on GitHub.

         I'm having issues with my self-hosted Mempool instance. Can you help?

    Many issues can be solved by making sure you're running the latest official release of Mempool, and making sure both Bitcoin and Electrum are fully synced.

    Keep in mind that Electrum can take another day or so to sync after Bitcoin Core has finished syncing.
